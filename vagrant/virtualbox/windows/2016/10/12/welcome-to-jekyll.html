Installing VirtualBox 5 on Windows 10

Installing VirtualBox on Windows 10 is pretty straightforward. There are a number of editions of Windows 10. Two of the more common editions include:

  • Windows 10 Home
  • Windows 10 Pro / Windows 10 Enterprise

With Windows 10 Pro, a hypervisor is built in and included with this edition of Windows. It is not included within Windows 10 Home. This hypervisor is referred to as HyperV.

By default, with a standard installation of Windows 10 Pro, HyperV is not enabled. HyperV can enabled/disabled via the Windows Features dialog.

Only one hypervisor can be active at any given time within Windows. Its important that HyperV be disabled when working with Oracle VirtualBox.
